# Privacy Policy

**Attribute Intelligence Agent** collects, stores, and transmits no data.

## What this plugin is

The plugin is a set of Markdown instruction files — a skill definition and a subagent definition. It contains no executable code of its own, makes no network calls, and has no server component. It relies entirely on Claude's built-in code execution to analyze files inside your own session.

## Your catalog data

Product catalog files (CSV, Excel, etc.) you upload stay entirely within your Claude conversation and are handled under [Anthropic's privacy policy](https://www.anthropic.com/legal/privacy). The plugin does not copy, forward, log, or retain your data, and the author (GAMI Solutions) has no access to it.

## Credentials and data access

None required. The plugin requests no API keys, database credentials, or warehouse connections, and cannot reach any system beyond the file you attach to the conversation.

## Third parties

No data is shared with the plugin author or any third party.

## Changes

Any change to this policy will be committed to this repository and reflected in the date below.
